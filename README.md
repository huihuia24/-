# 网页定时管理器使用说明

本文档将详细介绍网页定时管理器用户脚本的安装步骤、功能模块、操作方法及注意事项，帮助你快速上手使用该工具。

## 一、脚本概述

网页定时管理器是一款适用于油猴`（Tampermonkey）`等脚本管理器的工具，核心支持**当前网页定时操作**和**多网址定时跳转**两大核心功能，搭配可拖拽、明暗主题切换等优化特性，满足网页定时管理的多样化需求，且操作界面简洁直观，适配不同使用场景。

## 二、安装步骤

1. 确保浏览器已安装**油猴`（Tampermonkey）`** 等扩展插件，支持 Chrome、Edge、Firefox 等主流浏览器，可通过浏览器应用商店搜索安装。

   推荐：

   - [脚本猫](https://scriptcat.org/zh-CN)
   - [油猴](https://www.tampermonkey.net/index.php?locale=zh)

2. 打开**脚本猫**、**油猴**

   - 脚本猫：进入脚本**管理界面( 已安装脚本)**，点击**新建脚本**，点击**新建普通脚本**，进入**脚本编辑器**
   - 油猴：进入油猴**管理面板**，**点击`+`号**，进入**脚本编辑器**

3. 复制：在本页面打开[网页定时管理器.user.js](https://github.com/huihuia24/-/blob/main/网页定时管理器.user.js)，`CTRL+A`将代码全选复制到**脚本猫**、**油猴**

4. 点击脚本编辑器的保存按钮（或使用快捷键`Ctrl+S`），脚本自动生效并适配所有网页。

## 三、界面介绍

脚本加载后默认以**最小化状态**显示在网页右上角，核心界面分为最小化界面和完整功能界面，支持自由切换与拖动。

1. 最小化界面

   - 显示内容：仅展示倒计时图标与时间（默认显示 `00:00:00`）。
   - 核心操作：双击该区域可展开完整功能界面；按住鼠标可拖动至网页任意位置，位置会自动保存。


2. 完整功能界面

   界面默认展开后位于右上角，支持按住顶部区域拖动。整体分为 3 个核心区域：

   | 区域       | 功能说明                                                     |
   | ---------- | ------------------------------------------------------------ |
   | 顶部控制栏 | 左侧显示工具名称，右侧包含**主题切换按钮**（太阳 / 月亮图标）和**最小化按钮**（压缩图标），点击最小化按钮可返回最小化状态 |
   | 标签切换区 | 包含 “当前网页定时” 和 “多网址定时跳转” 两个标签，点击可切换对应功能页面，切换不影响定时任务运行 |
   | 功能操作区 | 对应两个标签的具体设置项、操作按钮和状态提示，包含输入框、按钮、任务列表等交互元素 |

## 四、功能操作指南

### （一）当前网页定时

该模块用于对当前打开的网页设置定时，到期后执行指定操作，步骤如下：

1. 在完整界面切换至 “当前网页定时” 标签。
2. **设置定时时长**：在小时、分、秒三个大字体输入框中输入对应数值（支持 0 及以上合理范围，分钟和秒最大值为 59）。
3. 选择结束后操作
   - 跳转至空白页：默认选项，定时结束后自动跳转到浏览器空白页。
   - 跳转至指定网页：选中对应单选框后，下方会显示网址输入框，输入目标网址（支持 http/https 协议，未输入协议时自动补充 https）。
4. 点击**开始定时**按钮启动定时，状态栏会显示剩余时间；如需终止，点击**停止定时**按钮即可。

### （二）多网址定时跳转

该模块支持添加多个网址，设置每个网址的停留时长，实现跳转后倒计时、倒计时结束自动跳转到下一个网址的循环操作，步骤如下：

1. 切换至 “多网址定时跳转” 标签。
2. 添加单个定时任务
   1. 在 “网址” 输入框中输入目标网址。
   2. 填写该网址的停留时长（跳转后开始计时，小时 / 分 / 秒输入规则同上）。
   3. 可选设置：勾选 “在新标签页打开”，跳转时不会关闭当前页，而是在新标签打开目标网址。
   4. 点击**添加到列表**，任务会以 “网址 + 时长” 的形式显示在下方任务列表中。
3. 管理任务列表
   - 删除单个任务：点击任务右侧的关闭图标，可移除该任务。
   - 清空列表：点击**清空列表**按钮，可一键清空所有任务（无弹窗提示，直接清空）。
4. 启动多网址跳转
   1. 确认任务列表无误后，点击**开始执行任务**。
   2. 脚本会按列表顺序跳转至第一个网址，跳转完成后开始倒计时。
   3. 倒计时结束后自动跳转至下一个网址，直至所有任务执行完毕。
   4. 执行过程中如需终止，点击**停止执行**按钮即可。

## 五、特色优化功能

1. 明暗主题切换

   点击顶部控制栏的主题按钮，可切换浅色 / 深色主题。主题偏好会自动保存，切换后字体、图标对比度自适应调整，确保黑色背景下正常显示，白色背景下样式不异常。

2. 位置记忆

   无论是完整界面还是最小化界面，拖动后的位置都会被保存，刷新网页或切换页面后，界面仍会显示在之前调整的位置。

3. 流畅交互优化

   拖动界面时，UI 会流畅跟随鼠标移动，无延迟卡顿；定时输入框字体放大后不会溢出界面，输入操作无异常。

4. 任务状态持久化

   多网址跳转过程中，即使跳转新网页，任务状态也会保存，不会出现进程丢失；刷新网页后可恢复未完成的定时任务。


## 六、注意事项

1. 网址输入时尽量填写完整（如包含 http/https），若未填写协议，脚本会自动补充 https 协议，确保正常跳转。
2. 定时时长需设置大于 0 的数值，否则会提示 “请设置有效的时间”，无法启动定时。
3. 多网址跳转时，若选择 “在新标签打开”，会持续生成新标签页，建议合理规划任务数量，避免标签页过多。
4. 脚本仅在安装了油猴插件的浏览器中生效，且适配所有网页，无需单独配置适配规则。
